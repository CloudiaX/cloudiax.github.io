Natural User Interfaces

Real-world Objects Interaction with Microsoft PixelSense Table

In 2013 I engaged in the NUI project “Real-world Objects Interaction with Microsoft PixelSense Table” as a research assistant. We developed an application using the optical sensor of Microsoft PixelSense Table to recognize the phone on the table. By placing their mobile phones on the PixelSense Table, users could “shake” their photos out of the phone onto the table screen to share them with friends.

Sketch to Motion: A Sketch-based User Interface for NAO robot Motion Design

To explore more natural and interactive ways, in the project “Sketch to Motion: A Sketch-based User Interface for NAO robot Motion Design”, I presented a sketch-based user interface assisting users in designing NAO robot motions by sketching stick figure postures.

Uup

After doing a lot of research on natural user interfaces, I tried to solve some real-world problems. Seeing so many drivers distracted by the GPS map, I developed a mobile device called “Uup” that could use the reflection on the vehicular windscreen as a display screen.

Group efforts to further polish our product won us a 50,000-yuan funding from “Tsinghua Innovation Plus”, and I co-founded a start-up in 2015.


Motion Sensing Augmented Reality Game Design and Development

I developed a motion-sensing augmented reality game as my undergraduate thesis project in 2011. In this game, the player could use gestures (PC version) or motions (Mobile version) to control the panel to play a ping-pong game with computer in an augmented reality environment. I also designed all the graphic elements in this project.

Music-Driven Box (LEGO Project)

We used the sound sensor of LOGO robot kit to build an interactive music box which would rotate in circle when it “heard” music.
